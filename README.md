# wfetch-rs

## Wfetch-rs is a Rust command-line application for obtaining basic weather information

### How to install

Clone the repo:

```sh
git clone https://github.com/EuCaue/wfetch-rs.git
```

Change directory:

```sh
cd wfetch-rs/
```

Compile:

```sh
cargo build --release
```

Change directory and run:

```sh
cd target/release && ./wfetch
```
